# harry-potter-api

## Inicializando o projeto
- Para rodar o projeto incialmente deve ter docker instalado na máquina e executar o comando `docker-compose up -d`, para inicializar a infraestrutura base;
- Executar o comando `npm install` com finalizade de instalar os pacotes de dependencias;

## Executando
- Para executar o projeto basta rodar o comando `npm start`;

## Rodando os testes
- Após fazer os setup do projeto, iniciando o docker e instalado as dependencias basta rodar o comando `npm test`;

## Documetação Api
- A api conta com uma documentação gerada pelo swagger, para acessar a mesma basta colocar o endereço `http://localhost:3300/api-docs/`no seu browser, todos os 4 endpoin`s estão funcionais;
